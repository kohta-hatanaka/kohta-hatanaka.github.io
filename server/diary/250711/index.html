July 11, 2025

久しぶりに古いdesktop pcを起動した。
高校の時に買ったCDを聞きたくなって、CD playerという遺物があるのはdesktopだけだったので引っ張り出してきた。
確か中学上がる時にお年玉と親の折半で買ってもらったやつで、もう10年も前の骨董品だ。

中にはまだwindows10が入っていて、更新したらwindows11が適応できないときた。

Windows media playerとかいう、これまた時代遅れのデザインのソフトで、CDを取り込む。
ついでにブックマークとエクスプローラを覗いたら、大学3年くらいで時が止まっていた。
役員会の資料を見て懐かしく思うようになってしまった。

曲を流していて、なんだかセンチメンタルな気分になってきた。

と、感傷に浸っていたら友人から山岡家に行くぞと連絡が入った。
明日はデートだというのに全くもう。
仕方がないから付き合ってやることにする。

もうスペック的には過去のものだが、このpcは青春の思い出なのかもしれない。 タイムマシンとしてもうしばらく持っておくとしよう。
